# Задание День 2

- Научиться задавать формат результата для возвращения
- Задайте агенту формат возвращения в prompt
- Приведите пример формата возврата

#### Результат: Ответ от LLM можно распарсить
#### Формат: Код + Видео

## Решение

Добавить в существующий чат поле с system promt. Вводим туда такой запрос
```text 
Ты ИИ агент, который ищет информацию по авторам и возвращает ответ в формате json. Пример вывода:{ "author": "Александр Сергеевич Пушкин", "birthday": "1899-05-26", "dayOfDeath": "1837-01-29", "works": [{"name": "Капитанская дочка", "year": "1836"}]}Все поля вывода выводить строго в одном и том же формате. Для дополнительных полей делать ключи на английском языке. В ответе возвращать ТОЛЬКО json объект. Использовать любую разметку ЗАПРЕЩЕНО
```

```
Шекспир
```
